# 小薇 AI 助手 Demo 合集

两个 AI 助手的手机端交互 Demo，可直接在浏览器中打开使用。

## 📱 Demo 列表

| Demo | 说明 | 预览 |
|------|------|------|
| [小薇生活助手](xiaowei-life-assistant/) | 航班查询 AI 助手 - Flight Agent Demo | 手机模拟器 |
| [小薇找工作](xiaowei-job-agent/) | AI 求职助手 - Job Agent Demo | 手机模拟器 |

## 🚀 使用方式

### 在线使用
访问 GitHub Pages 链接即可使用（部署后自动生效）。

### 本地使用
1. 下载整个项目
2. 双击对应文件夹中的 `index.html` 在浏览器中打开
3. 无需安装任何依赖

## 🛠 技术说明
- 纯 HTML/CSS/JS 实现
- 无外部依赖，离线可用
- 手机模拟器 UI 展示
